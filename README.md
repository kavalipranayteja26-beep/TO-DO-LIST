# 🌿 Pastel Day Planner & Tracker

A stunning, minimalist, and highly interactive daily planning application built with modern web aesthetics. Designed with a **Pastel Green & White** palette, this app helps you manage your tasks, water intake, and medicine schedule with ease.

![App Dashboard Preview](https://github.com/kavalipranayteja26-beep/TO-DO-LIST/raw/main/preview.png) *(Placeholder: You can replace this later with a screenshot)*

## ✨ Features

-   **Interactive Setup**: Plan your upcoming day with a step-by-step guided setup.
-   **Habit Tracking**: Dedicated tracker for **Water Intake** with interactive glass animations.
-   **Medicine Management**: Schedule your medications and get reminders.
-   **Smart Notifications**: Native browser alerts to remind you when a task or medicine is due, with 10-minute early warnings.
-   **Progress Analytics**: A circular progress ring that dynamically updates as you complete your goals.
-   **Persistent Data**: Automatically saves your plan to your browser's local storage.

## 🎨 Design Philosophy

-   **Pastel Palette**: Mint Green and Soft Whites for a relaxing user experience.
-   **Glassmorphism**: Modern UI using semi-transparent glass panels with backdrop filters.
-   **Micro-animations**: Smooth transitions, hover effects, and animated checkmarks for satisfying task completion.

## 🚀 Getting Started

To run this application locally, ensure you have [Node.js](https://nodejs.org/) installed:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/kavalipranayteja26-beep/TO-DO-LIST.git
    cd TO-DO-LIST
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open in your browser**: Navigate to `http://localhost:5173/`.

## 📜 How to Use

1.  **Initial Setup**: When you first open the app, you'll be greeted with the **Plan Your Day** screen. Add your scheduled tasks and medications.
2.  **Tracking**: Once the plan is finalized, use the **Dashboard** to mark items as complete. Click on the water glass icons each time you drink water.
3.  **Notifications**: Ensure you **Allow Notifications** in your browser so the app can alert you even if the tab is in the background.
